MEUS PROJETOS

Aroma e Café

O Aroma & Café é uma aplicação web completa desenvolvida com Python e o framework Flask. O projeto simula uma cafeteria, oferecendo um sistema de gestão de estoque para a equipe interna e uma vitrine de e-commerce para os clientes.

Python - Flask - SQLite - HTML - CSS

FIGHTING FROG

FIGHTING FROG, um jogo de combate frenético e desafiador onde você controla um sapo lutador em uma cidade infestada de moscas inimigas. Seu objetivo é sobreviver o máximo de rodadas possível, derrotando hordas de insetos e aumentando sua pontuação.

Python - Pygame

PDF TO AUDIOBOOK

Um script em Python simples e leve que converte o texto de um arquivo PDF inteiro em um arquivo de áudio no formato MP3. Perfeito para transformar seus documentos em audiobooks sem a necessidade de dependências externas complexas.

Python - PyPDF2 - pyttsx3

THE CONJURING

Um projeto de carrossel interativo e responsivo que explora o universo cinematográfico de Invocação do Mal. Este projeto foi desenvolvido com foco na prática de HTML, CSS e JavaScript, criando uma experiência sincronizada onde a imagem, a descrição e os botões de navegação mudam simultaneamente.

HTML - CSS - JS

TO-DO LIST

Um aplicativo simples e moderno de lista de tarefas, construído com Python e a biblioteca customtkinter. Este projeto permite que você gerencie suas tarefas diárias com uma interface limpa, incluindo funcionalidades como adicionar, remover e visualizar itens.

Python - Customtkinter

SISTEMA BIBLIOTECA

Este projeto visa demonstrar a criação de um sistema de CRUD (Create, Read, Update, Delete) com persistência de dados em um banco de dados MySQL, utilizando o conceito de Programação Orientada a Objetos (POO) para organizar o código em classes coesas.

Python - MySQL - mysql-connector
